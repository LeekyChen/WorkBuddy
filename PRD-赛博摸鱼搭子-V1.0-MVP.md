# 产品需求文档（PRD）：赛博摸鱼搭子（Cyber Slacker）V1.0

## 1. 产品概述（Product Overview）

### 1.1 背景与痛点

当前桌面端的 AI 工具严重同质化，都在极力卷“生产力”（代码补全、文档生成），仿佛一个不知疲倦的数字监工，给本就高压的开发者带来了额外的精神负担。开发者在遇到 Bug、编译漫长或纯粹想发呆时，缺乏一个轻量级、懂梗、能提供情绪价值的桌面陪伴物。

### 1.2 产品定位

一个无需联网、不侵犯隐私、极低能耗的桌面级 AI 宠物。它不仅是一个桌宠，更是一个拥有“人格”的赛博反卷先锋。

### 1.3 目标用户

- 经常面对报错日志发呆的后端/全栈开发者。
- 需要在长时间 Coding 中寻找情绪出口的极客。

## 2. 核心功能需求（Functional Requirements）

MVP 阶段我们将系统拆分为三个核心模块：**感知层**、**思考层**、**表现层**。

### 2.1 感知层（The Observer）- 绝对不使用截屏

系统需要以极低的开销在后台静默运行，获取用户当前状态。

| 需求点 | 描述 | 实现方式建议 | 优先级 |
|---|---|---|---|
| 进程嗅探 | 识别用户当前的前台应用，判断是在“卷”还是在“摸鱼”。 | 轮询系统活动窗口（如识别到 IDE、终端、浏览器、Steam 等进程）。 | P0 |
| 高频词监听 | 识别开发者是否陷入了困境或进行了高危操作。 | 监听终端 history 文件的尾部追加（例如连续出现 `error`, `exception`, `git push -f`）。 | P1 |
| 目录静默监控 | 感知用户的代码产出频率。 | 监听目标工作区的文件 Save 事件。如果一小时没保存，触发“发呆”提醒。 | P2 |

### 2.2 思考层（The Brain）- 纯本地驱动

结合感知层的数据，通过 LLM 生成符合人设的吐槽或关怀。

| 需求点 | 描述 | 触发逻辑示例 | 优先级 |
|---|---|---|---|
| 基于状态的 Prompt 组装 | 将系统信息转化为自然语言上下文输入给模型。 | `[System: 用户正在 VSCodium/Cursor 中频繁切换，且终端刚报错。请用阴阳怪气的资深程序员口吻吐槽一句，20字以内。]` | P0 |
| 随机主动搭话 | 打破死寂，提供惊喜感。 | 设置定时器，每隔 45-90 分钟随机触发一次闲聊（如：“到饭点了，今天吃啥？”）。 | P1 |
| 极简对话接口 | 用户可以直接点击宠物输入文本对骂或聊天。 | 弹出一个极简的 Input 框接收输入，传入 LLM 并返回气泡回复。 | P1 |

### 2.3 表现层（The UI）- 沉浸式陪伴

UI 必须足够轻量，不能打断用户的正常工作流。

| 需求点 | 描述 | 优先级 |
|---|---|---|
| 无边框与透明背景 | 宠物本体和聊天气泡需要悬浮在所有窗口之上，无系统原生窗口边框。 | P0 |
| 永远置顶与鼠标穿透 | 保持在屏幕角落，且支持配置“鼠标穿透”（点击时直接点到下层代码，不遮挡视线）。 | P0 |
| 极简动图反馈 | 待机、敲键盘、生气、睡觉等基础 GIF 帧切换。 | P1 |

## 3. 非功能性需求（Non-Functional Requirements）

- **极低资源占用（P0）**：尤其是在 ARM64 架构的设备上运行时，绝不能引发风扇狂转或电池剧烈消耗。除了模型推理瞬间，常驻内存需控制在 **50MB** 以内。
- **隐私安全绝对隔离（P0）**：所有的进程名、终端历史只存在于本地内存中。严禁任何形式的云端遥测（Telemetry）或数据上报。
- **冷启动速度（P1）**：点击图标后 **2 秒内**必须出现在桌面上。

## 4. 技术栈建议（Technical Stack）

为了实现极速开发且保证跨平台（尤其是针对 macOS 的良好支持），建议采用以下架构：

### 4.1 宿主框架

- **方案 A（推荐）**：Wails（Go 语言后端 + 前端轻量级渲染）。Go 极其擅长处理高并发的进程轮询和文件监控逻辑，且打包体积小，性能绝佳。
- **方案 B**：Python + PyQt/PySide。Python 生态中有丰富的系统监控库（如 `psutil`, `watchdog`），写起来最快。

### 4.2 AI 引擎

- 接入本地运行的 **Ollama API**。
- 让用户自己在本地跑一个轻量级模型（比如 **Qwen-0.5B** 或 **Llama-3-8B**），脚本只负责发送 HTTP 请求。

### 4.3 UI 素材

- itch.io 上有很多免费开源的像素风 2D 宠物 Sprite Sheet。

## 5. MVP 周末开发冲刺计划（Weekend Hackathon Plan）

### Day 1

- 搭骨架：跑通无边框、透明、置顶的桌面窗口。
- 写出操作系统的进程读取脚本和文件监控逻辑。
- 搞定简单的 UI 点击和拖拽移动。

### Day 2

- 注灵魂：对接本地 Ollama API，调试 Prompt 模板（这个最花时间，决定了宠物有多好玩）。
- 将感知层的数据作为变量塞进 Prompt。
- 完成打包（构建可执行程序），准备开源发布。
